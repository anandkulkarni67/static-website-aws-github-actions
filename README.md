# A Static website deployed to AWS cloudfront served from S3 with CICD implementation using AWS CodePipeline

## download and install dependencies
```
npm install
```

## Compiles and hot-reloads the project for development
```
npm run serve
```

## Compiles and minifiesthe project for production deployment
```
npm run build
```

## AWS Stacks creation
```
Recommended Order:
1. Application
2. Security
```

## GitHub integration
```
1. Go to AWS Console -> AWS CodePipeline -> Settings ( Left Side-panel ) -> Connections
2. Click on the pending connection.
3. Follow the prompts on screen to establish connection between AWS and Github repository/s.
```
